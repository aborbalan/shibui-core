# Molecules Documentation

This section contains detailed documentation for **Molecule** UI components.

Molecules are simple functional groups formed by combining atoms. They serve a single, well-defined purpose and represent the first level of composition in the design system.

## Available Molecules

-   [Breadcrumb](./breadcrumb.md): Navigation component showing the current page's location within the site hierarchy.
-   [Button Group](./button-group.md): Collection of related buttons grouped together.
-   [Form Field](./form-field.md): Complete form input with label, input, and error message.
-   [Input](./input.md): Text input field with additional functionality.
-   [Lib Alert](./lib-alert.md): Alert component for displaying informational, warning, error, or success messages.
-   [Modal](./modal.md): Overlay window for displaying content above the main interface.
-   [Multiselect](./multiselect.md): Dropdown component allowing multiple option selection.
-   [Profile Card](./profile-card.md): User profile display combining avatar, name, and additional information.
-   [Select](./select.md): Dropdown component for single option selection.
-   [Tabs](./tabs.md): Tabbed navigation component for organizing content into separate views.
