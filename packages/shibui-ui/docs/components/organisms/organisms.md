# Organisms Documentation

This section contains detailed documentation for **Organism** UI components.

Organisms are complex, standalone interface sections formed by combining molecules and atoms. They represent distinct sections of an interface and often function like "mini-applications" with their own internal logic and state management.

## Available Organisms

-   [Dialog](./dialog.md): Modal window for displaying content and requiring user interaction.
-   [Lib Auth Form](./lib-auth-form.md): Authentication form component for user login and registration.
-   [Lib List](./lib-list.md): Complex list component with advanced features like filtering, sorting, and pagination.
-   [Sidebar](./sidebar.md): Navigation panel typically positioned on the side of the interface.
